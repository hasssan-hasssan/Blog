# venv
### if you wanna run this project by venv do these steps : 
```shell

$ git clone https://github.com/hasssan-hasssan/Blog.git

$ cd Blog

$ python -m venv venv

$ venv/scripts/activate.bat

$ pip install -r requirements.txt

$ python manage.py runserver

```
